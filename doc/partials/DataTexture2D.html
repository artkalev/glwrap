DataTexture2D

DataTexture2D

Draw TypedArray as 2D texture. also used in Framebuffer2D.

Constructor

new DataTexture2D(data, width, height, internalFormat, sourceFormat, pixelType)

Arguments:
  1. data (ArrayBufferView)
  2. width (Number)
  3. height (Number)
  4. internalFormat (String)  gl pixel format. default: RGBA
  5. sourceFormat (String)  gl pixel format. default: RGBA
  6. pixelType (String)  gl pixel type. default UNSIGNED_BYTE

Methods

init(gl)

Creates the WebGLTexture instance. There is no need to call this manually in most cases. It is called from BaseTexture2D#setActive if needed.
Arguments:
  1. gl (WebGLRenderingContext)
  • BaseTexture2D#init

setActive(gl, textureUnit)

Initializes and updates this texture if necessary and then binds it to given texture unit. This is used internally when ShaderProgram sets a texture uniform.
Arguments:
  1. gl (WebGLRenderingContext)
  2. textureUnit (Number)
  • BaseTexture2D#setActive

update(gl)

Updates webgl texture data and parameters.
