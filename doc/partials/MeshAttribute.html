MeshAttribute

Mesh Attribute

Constructor

new MeshAttribute(name, data, size, type, normalized, usage)

Arguments:
  1. name (String)  attribute name
  2. data (ArrayBufferView)  typed array of data for the attribute
  3. size (Number)  size of data in array. 2 for vec2, 3 for vec3, etc...
  4. type (String)  datatype matching the typed array datatype. 'FLOAT', 'UNSIGNED_BYTE', etc
  5. normalized (Boolean)  wether the data is normalised to 0-1 range for glsl.
  6. usage (String)  usage hint for gl. 'STATIC_DRAW' or 'DYNAMIC_DRAW'.

Methods

bind(gl, program)

initializes buffer. Internal method. Should not be called manually!
Arguments:
  1. gl (WebGLRenderingContext)  gl context.
  2. program (ShaderProgram)  shaderProgram instance.

initializes buffer. Internal method. Should not be called manually!
