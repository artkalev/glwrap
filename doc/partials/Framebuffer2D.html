Framebuffer2D

Framebuffer2D

Contains color and depth rendering target buffers for WebGL. Essential for render-to-texture implementations.

Constructor

new Framebuffer2D(width, height, useColor, useDepth)

Arguments:
  1. width (Number)  width of the framebuffer in pixels
  2. height (Number)  height of the framebuffer in pixels
  3. useColor (Boolean)  wether to create buffer for color data (cannot be changed after creation)
  4. useDepth (Boolean)  wether to create buffer for depth data (cannot be changed after creation)

Methods

init(gl)

initializes buffers. internal method, this should not be called manually!
Arguments:
  1. gl (WebGLRenderingContext)  gl context

setActive(gl)

binds this framebuffer and sets up gl viewport. also initializes and/or updates buffers if necessary.
Arguments:
  1. gl (WebGLRenderingContext)

update(gl)

updates buffers. Automatically called by this.setActive() if needsUpdate == true
Arguments:
  1. gl (WebGLRenderingContext)  gl context
