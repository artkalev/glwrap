Mat4

4 x 4 Matrix

Constructor

new Mat4()

Source:
  • mat4.js, line 12
Creates new identity matrix.
Example
// creates a transformation matrix with translation (3,5,4)
let modelMatrix = new Mat4();
modelMatrix.trs(
     new Vec3(3,5,4),
     new Quat(),
     new Vec3(1,1,1)
);
